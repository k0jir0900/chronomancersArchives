Authentication

All API requests require an API key passed in the request header.

HeaderValueRequired
X-API-Key Your personal API key Yes
Generate or regenerate your API key from My Profile. Keys follow the format ca_<64 hex chars>.

Base URL

{{ request.host_url.rstrip('/') }}/api/v1

All endpoints are relative to this base URL.

Endpoints

GET /api/v1/cdu

Returns a paginated list of all CDU (Correlation Detection Unit) records. Optionally filter by company.

Query Parameters
ParameterTypeDefaultDescription
company string - Filter by exact company name. Omit to return all companies.
page integer 1 Page number. Starts at 1.
limit integer 100 Records per page. Min: 1, Max: 500.
Response 200 OK
{
  "total":  412,
  "page":   1,
  "limit":  100,
  "pages":  5,
  "data": [ ... ]
}

Examples

cURL - All records
curl -X GET "{{ request.host_url.rstrip('/') }}/api/v1/cdu" \
     -H "X-API-Key: ca_your_api_key_here"
cURL - Filter by company with pagination
curl -X GET "{{ request.host_url.rstrip('/') }}/api/v1/cdu?company=Acme+Corp&page=2&limit=50" \
     -H "X-API-Key: ca_your_api_key_here"
Python (requests)
import requests

API_KEY = "ca_your_api_key_here"
BASE    = "{{ request.host_url.rstrip('/') }}/api/v1"

resp = requests.get(
    f"{BASE}/cdu",
    headers={"X-API-Key": API_KEY},
    params={"company": "Acme Corp", "limit": 500}
)
resp.raise_for_status()
data = resp.json()

print(f"Total records: {data['total']}")
for cdu in data["data"]:
    print(cdu["rule_name"], cdu["action_type"], cdu["created_at"])
